# Predicción de Demanda en E-commerce

Este proyecto implementa modelos de Machine Learning y Deep Learning para predecir la demanda en un entorno de comercio electrónico, utilizando el dataset Predict Future Sales.

## Estructura del proyecto
- `ModeloPredictivoVentasEcommerce.ipynb`: Notebook principal con análisis exploratorio, entrenamiento y evaluación.

## Instalación y ejecución
1. Clonar el repositorio:
```bash
git clone https://github.com/estebangarviso/ecommerce-demand-prediction.git
````

2.  Abrir el notebook en Google Colab.
3.  Instalar dependencias:

```bash
!pip install scikit-learn xgboost tensorflow shap plotly pandas
```

4.  Ejecutar las celdas en orden.

## Objetivo

Predecir la demanda mensual de productos para optimizar inventarios y reducir costos.

## Dependencias

*   Python 3.10+
*   scikit-learn
*   xgboost
*   tensorflow
*   shap
*   plotly
